# LykOS

## Overview
This is the main repository of LykOS, an open source hobbyist operating system targeting 64-bit architectures.

## Features
- x86_64 & aarch64 support
- Multithreading & SMP
- Buddy allocator, slab allocator, & virtual memory

## Roadmap
- Kernel module system
- POSIX compatibility
- File system & disk drivers
- Network stack
- Porting software
